# 🛒 Retail GenAI with LangGraph

## 📌 Overview
This project is an AI-powered retail assistant that helps businesses make smarter decisions using GenAI.

## 🚀 Features
- Price Optimization
- Demand Analysis
- Stock Recommendation
- AI-powered Query Handling

## 🧠 Tech Stack
- LangChain
- LangGraph
- FastAPI
- FAISS

```bash
pip install -r requirements.txt
